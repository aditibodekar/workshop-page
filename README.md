AI Robotics Workshop Landing Page

A full-stack web application for managing and showcasing an AI Robotics Workshop. This project includes a frontend built with React, Vite, and Tailwind CSS, along with a backend powered by Node.js, Express, TypeScript, and MongoDB.

Project Structure

workshop-landing/
├── frontend/   # React + Vite + Tailwind CSS
├── backend/    # Node.js + Express + TypeScript + MongoDB

Features

Responsive workshop landing page
Interactive UI with Tailwind CSS
Backend API integration
MongoDB database support
Form handling for registrations
Environment variable support
